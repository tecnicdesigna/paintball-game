# 🎨 Paintball Online Game

## বন্ধুদের সাথে Online খেলার জন্য Railway-তে Deploy করুন

### ধাপ ১ — GitHub Account বানান
- https://github.com এ যান
- Sign up করুন (Free)

### ধাপ ২ — এই folder টা GitHub-এ Upload করুন
1. GitHub এ New Repository বানান (নাম: paintball-game)
2. Upload files button চাপুন
3. server.js, package.json এবং public/ folder টা drag করে দিন
4. Commit changes চাপুন

### ধাপ ৩ — Railway-তে Deploy করুন
1. https://railway.app এ যান
2. GitHub দিয়ে Login করুন
3. "New Project" → "Deploy from GitHub repo" চাপুন
4. আপনার paintball-game repo select করুন
5. Deploy হয়ে যাবে! (২-৩ মিনিট লাগবে)

### ধাপ ৪ — Link পান
- Railway dashboard এ আপনার app এর URL দেখাবে
- যেমন: https://paintball-game-production.up.railway.app
- এই link বন্ধুদের WhatsApp/Messenger এ share করুন!

### খেলার নিয়ম
- নাম দিয়ে Join করুন
- রঙিন পানির বল ছুড়ে শত্রুকে মাখিয়ে দিন!
- বেশি রঙ লাগলে HP শেষ হয়
- মরলে ৩ সেকেন্ড পর আবার জন্ম নেবে
- কেউ না থাকলে Bot খেলবে

### Controls
- Mobile: Joystick (বাম) + Fire Button (ডান)
- PC: WASD + Mouse + Click
